# OhMyStock

**Step by step build a US stock analytics platform by myself.**

## What's OhMyStock

OhMyStock is a medium to low frequency trading analytics platform. We analyze the stock data for each day.

## Technology Stack

- Laravel
- MangoDB
- MariaDB
